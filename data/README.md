# Dados do Projeto
   
   ## Dataset: processed.cleveland.data
   
   **Fonte:** UCI Machine Learning Repository
   
   ### Descrição dos Atributos:
   
   1. age - Idade do paciente
   2. sex - Sexo (1=M, 0=F)
   3. cp - Tipo de dor no peito (1-4)
   4. trestbps - Pressão arterial em repouso
   5. chol - Colesterol sérico
   6. fbs - Glicemia em jejum > 120 mg/dl
   7. restecg - Resultados ECG em repouso
   8. thalach - Frequência cardíaca máxima
   9. exang - Angina induzida por exercício
   10. oldpeak - Depressão ST induzida
   11. slope - Inclinação do segmento ST
   12. ca - Número de vasos principais
   13. thal - Talassemia
   14. target - Diagnóstico (0=ausente, 1-4=presente)
